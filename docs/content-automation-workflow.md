### 阶段3：内容创作（人工+AI辅助）

#### AI辅助写作流程
```
1. 输入关键词和大纲
2. AI生成初稿（ChatGPT/Claude）
3. 人工润色（添加品牌调性）
4. 添加产品植入点
5. 优化SEO（标题、描述、关键词密度）
```

#### 内容检查清单
- [ ] 标题包含核心关键词
- [ ] 首段出现关键词
- [ ] 正文关键词密度2-3%
- [ ] 小标题使用H2/H3
- [ ] 图片有alt属性
- [ ] 内部链接3-5个
- [ ] 字数>800字
- [ ] 原创度>80%

### 阶段4：自动发布（自动化）

#### GitHub Actions自动发布
```yaml
# .github/workflows/publish-content.yml
name: Publish Content
on:
  schedule:
    - cron: '0 9 * * 1'  # 每周一9点
jobs:
  publish:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Build and Deploy
        run: |
          npm run build
          vercel --prod
```
