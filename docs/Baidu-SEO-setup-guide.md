# 百度搜索资源平台配置指南

## 🎯 目标
让百度收录爱美舜网站，提升百度搜索排名

---

## 📱 步骤1：注册百度账号（1分钟）

1. 访问：https://ziyuan.baidu.com
2. 使用百度账号登录（或注册新账号）
3. 建议使用企业邮箱注册

---

## 🔧 步骤2：添加网站（3分钟）

### 1. 点击"添加网站"

### 2. 选择协议头
```
选择：https://
```

### 3. 填写网站地址
```
输入：www.aimeishun.cn
```

### 4. 选择站点领域
```
选择：美妆/护肤/个人护理
```

---

## ✅ 步骤3：验证网站所有权（5分钟）

### 方式1：文件验证（推荐）

1. 下载验证文件
```
点击"文件验证" → 下载验证文件
文件名类似：baidu_verify_xxxx.html
```

2. 上传验证文件
```
将文件上传到网站根目录：
public/baidu_verify_xxxx.html
```

3. 验证
```
点击"完成验证"
```

### 方式2：HTML标签验证

1. 复制meta标签
```html
<meta name="baidu-site-verification" content="code-xxxxxxxx" />
```

2. 添加到网站头部
```
修改 app/layout.tsx
在 <head> 中添加该meta标签
```

3. 部署后点击验证

---

## 🗺️ 步骤4：提交Sitemap（2分钟）

### 1. 进入Sitemap提交
```
左侧菜单：索引管理 → Sitemap
```

### 2. 添加Sitemap地址
```
输入：https://www.aimeishun.cn/sitemap.xml
```

### 3. 提交
```
点击"提交"
```

### 4. 状态检查
```
等待状态变为：正常
时间：1-7天
```

## ⚠️ 常见问题

### Q1：验证失败怎么办？
```
检查：
1. 文件是否上传到正确位置
2. URL是否能正常访问
3. 是否有CDN缓存（清除缓存）
4. 等待10分钟后重试
```

### Q2：多久能被收录？
```
新站：1-4周
老站更新：3-7天

加速方法：
1. 提交sitemap
2. 手动提交链接
3. 发布高质量内容
4. 外部链接引导
```

### Q3：收录了但没排名？
```
原因：
1. 内容质量不够
2. 关键词竞争大
3. 网站权重低

解决方法：
1. 持续发布原创内容
2. 优化页面SEO
3. 建立外链
4. 提升用户体验
```
