# Google Analytics 4 配置指南

## 🎯 目标
配置真实的GA4 Tracking ID，启用网站数据追踪

---

## 📱 步骤1：创建GA4账号（5分钟）

### 1. 访问 Google Analytics
- 网址：https://analytics.google.com
- 使用Google账号登录（建议用公司邮箱）

### 2. 创建账号
```
点击"开始衡量" → 填写：
- 账号名称：爱美舜
- 数据共享设置：按需求勾选（建议全选）
```

### 3. 创建属性
```
填写：
- 属性名称：爱美舜官网
- 报告时区：中国（东八区）
- 货币：人民币
```

### 4. 创建数据流
```
选择：网站
填写：
- 网站网址：https://www.aimeishun.cn
- 数据流名称：爱美舜官网
```

### 5. 获取Tracking ID
创建完成后，你会看到：
```
衡量ID：G-XXXXXXXXXX  ← 这就是Tracking ID
```

**复制这个ID，发送给我，我会配置到代码中**

## 📊 步骤2：配置转化事件（3分钟）

### 在GA后台设置：

1. 进入GA → 管理 → 事件 → 创建事件

2. 创建自定义事件：

| 事件名称 | 触发条件 | 说明 |
|---------|---------|------|
| `contact_click` | 点击咨询按钮 | 记录用户咨询意向 |
| `phone_click` | 点击电话 | 记录拨打电话 |
| `wechat_click` | 点击微信 | 记录添加微信 |
| `product_view` | 查看产品详情 | 记录产品兴趣 |
| `case_view` | 查看案例 | 记录案例兴趣 |

3. 标记为转化：
```
事件 → 对应事件右侧三点 → 标记为转化
```

---

## 🔍 步骤3：验证数据（2分钟）

### 实时报告检查：
1. 在GA后台点击"实时" → "概览"
2. 访问网站：https://www.aimeishun.cn
3. 应该能看到：
   - 用户数：1
   - 页面浏览：首页

### 如果看不到数据：
- 检查Tracking ID是否正确
- 检查代码是否部署
- 等待5-10分钟（数据有延迟）

---

## 📈 步骤4：设置关键指标（5分钟）

### 自定义报告：

1. **流量来源报告**
```
探索 → 自由形式
- 维度：会话来源
- 指标：用户数、会话数、平均停留时间
```

2. **转化漏斗**
```
探索 → 漏斗探索
- 步骤1：首页访问
- 步骤2：产品页浏览
- 步骤3：联系我们
- 步骤4：转化（咨询）
```

3. **用户行为报告**
```
探索 → 路径探索
- 查看用户浏览路径
- 发现流失节点
```

## ⚠️ 重要提示

### 隐私合规
```
GA收集用户数据，需要：
1. 在隐私政策中说明使用GA
2. 提供关闭追踪选项（可选）
3. 遵守《个人信息保护法》
```

### 数据保留
```
默认设置：
- 用户级数据：14个月
- 汇总数据：无限期

建议：保持默认即可
```

### 过滤内部流量
```
管理 → 数据流 → 网站 → 配置标记设置 → 显示全部
→ 定义内部流量 → 创建规则
- 规则名称：排除办公室IP
- IP地址：你的办公室IP
```

## 🚀 高级配置（可选）

### 目标受众群体
```
配置 → 受众群体 → 新建受众群体

1. 高意向用户
   - 条件：访问产品页 AND 停留时间>30秒
   
2. 回访用户
   - 条件：会话数>1
   
3. 移动端用户
   - 条件：设备类别 = 移动设备
```

### 自定义维度
```
配置 → 自定义定义 → 创建自定义维度

1. 用户类型
   - 范围：用户
   - 新用户 / 回访用户
   
2. 内容类型
   - 范围：事件
   - 产品 / 案例 / 博客
```
